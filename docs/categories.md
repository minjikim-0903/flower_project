# 상품 카테고리

## 꽃 종류별

| 분류 | 세부 품종 |
|---|---|
| 🌹 장미류 | 스탠다드 장미, 스프레이 장미 |
| 🌸 국화류 | 스탠다드 국화, 스프레이 국화, 소국 |
| 🌺 백합류 | 오리엔탈, 아시아틱, LA 하이브리드 |
| 💐 혼합 초화류 | 거베라, 카네이션, 튤립, 프리지아 |
| 🌿 그린류 (잎/줄기) | 안개꽃, 루스커스, 유칼립투스 등 보조 소재 |
| 🌻 계절화 | 해바라기, 수국, 작약 등 시즌별 꽃 |

---

## 용도별

| 용도 | 설명 |
|---|---|
| 웨딩 플라워 | 부케, 코사지, 테이블 장식용 |
| 근조 화환 | 장례식용 — 국화, 리시안셔스 등 |
| 행사/이벤트용 | 개업식, 축하 화환 |
| 일상 소비용 | 집 인테리어, 선물용 |

## 형태별

| 형태 | 설명 | 예시 |
|---|---|---|
| 절화 (Cut Flowers) | 줄기째 잘라 판매하는 꽃 | 장미, 튤립, 백합 등 |
| 분화 (Potted Plants) | 화분에 심어진 꽃 | 난초, 제라늄 등 |
| 구근 (Bulbs) | 구근 상태로 판매 | 튤립 구근, 수선화 구근 등 |
| 묘종 (Seedlings/Plugs) | 어린 모종 상태로 판매 | — |
