# 구매자

## 개요

구매자는 판매자가 등록한 상품을 탐색하고 주문할 수 있습니다.
로그인만 하면 즉시 구매자로 사용 가능합니다.

## 구매자 유형

| 유형 | 설명 |
|---|---|
| 꽃가게 운영자 | 꽃가게를 운영하며 상품 재입고 목적으로 구매 |
| 플로리스트 | 개인 플로리스트, 작업용 꽃 구매 |
| 웨딩플로리스트 | 웨딩·행사 전문 플로리스트 |
| 일반 구매 | 개인 소비 목적 구매 |
| 기타 | 직접 입력 |

> 구매자 유형은 니즈 파악 및 도매/소매 가격 적용 기준으로 활용됩니다.

## 주요 기능

### 탐색 및 검색
- 판매자 지점 목록 탐색
- 지점 검색
- 생화 / 나무 탭으로 상품 필터링

### 주문
- 소매 / 도매 주문 선택
- 배송 날짜 직접 지정
- 장바구니 담기 및 주문서 작성

### 주문 내역
- 주문 내역 조회
- 주문 상태 확인 (접수 → 준비 중 → 배송 중 → 완료)

## 주문 흐름

```
지점 탐색 → 상품 선택 → 장바구니
      ↓
주문서 작성 (배송날짜, 주문 유형 선택)
      ↓
주문 완료 → 판매자 처리 → 배송
```
