# 구매자

## 개요

구매자는 판매자가 등록한 상품을 탐색하고 주문할 수 있습니다.
로그인만 하면 즉시 구매자로 사용 가능합니다.

## 구매자 유형

| 유형 | 설명 |
|---|---|
| 꽃가게 운영자 | 꽃가게를 운영하며 상품 재입고 목적으로 구매 |
| 플로리스트 | 개인 플로리스트, 작업용 꽃 구매 |
| 웨딩플로리스트 | 웨딩·행사 전문 플로리스트 |
| 일반 구매 | 개인 소비 목적 구매 |
| 기타 | 직접 입력 |

> 구매자 유형은 니즈 파악 및 도매/소매 가격 적용 기준으로 활용됩니다.

---

## 화면 구조

### 하단 탭 메뉴

```
[ 홈 ]  |  [ 꽃 사전 ]  |  [ 설정 ]
```

---

### 홈 탭

판매자 지점 목록 탐색 및 상품 주문의 시작점

**주요 화면**
- 홈 메인 — 검색, 꽃 카테고리
- 지점 상세 — 지점 정보, 상품 목록
- 상품 상세 — 소매/도매 선택, 장바구니 담기
- 장바구니 — 담은 상품 목록, 수량 조정
- 주문서 — 배송 날짜 선택, 주문 유형 확인
- 주문 완료
- 주문 내역 — 전체 주문 목록
- 주문 상세 — 상태 확인 (접수 → 준비 중 → 배송 중 → 완료)

**주문 흐름**
```
홈 → 지점 탐색 → 상품 선택 → 장바구니
         ↓
주문서 작성 (배송날짜, 소매/도매 선택)
         ↓
주문 완료 → 판매자 처리 → 배송
```

---

### 꽃 사전 탭

> 기획 미정

---

### 설정 탭

계정 정보 및 역할 전환 관리

**메뉴 구성**
- 닉네임 변경
- 결제 목록
- 판매자로 전환 / 구매자로 전환
- 피드백 공간
- 커뮤니티 (미정)

**역할 전환 로직**

```
[설정] → 판매자로 전환 버튼
         ↓
현재 판매자 등록 여부 확인
         ↓
┌── 등록된 판매자 계정 있음 ──→ 판매자 모드로 즉시 전환
│
└── 판매자 미등록 ──→ 판매자 정보 입력 화면
                          ↓
                     사업자 정보 입력
                          ↓
                     관리자 승인 대기
                          ↓
                     승인 완료 → 판매자 모드 전환 가능
```

> 판매자 → 구매자 전환은 언제든 즉시 가능 (별도 승인 불필요)

---

## 주요 기능 요약

### 탐색 및 검색
- 판매자 지점 목록 탐색
- 지점 검색
- 생화 / 나무 탭으로 상품 필터링

### 주문
- 소매 / 도매 주문 선택
- 배송 날짜 직접 지정
- 장바구니 담기 및 주문서 작성

### 주문 내역
- 주문 내역 조회
- 주문 상태 확인 (접수 → 준비 중 → 배송 중 → 완료)
