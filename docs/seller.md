# 판매자

## 개요

판매자는 본인의 지점을 등록하고, 상품과 가격을 관리하며, 들어오는 주문을 처리합니다.
판매자 모드는 사업자등록증 보유자만 사용 가능합니다.

## 판매자 유형

| 유형 | 설명 |
|---|---|
| 꽃가게 운영 | 소매 중심의 꽃가게 운영자 |
| 도매소매업 | 도매 및 소매 병행 업자 |
| 기타 | 직접 입력 |

> 판매자 유형은 니즈 파악 및 가격 정책 적용을 위한 정보입니다.

## 주요 기능

### 지점 등록 및 관리
- 지점명, 주소, 대표 사진, 영업 상태 등록
- 지점 정보 수정

### 판매자 정보 등록 및 수정
- 판매자명, 연락처, 사업자 정보 등록/수정

## 주문 처리 흐름

```
구매자 주문 접수
      ↓
판매자 알림 수신
      ↓
주문 확인 → 처리 상태 변경 (접수 → 준비 중 → 배송 중 → 완료)
```

> 배송은 판매자가 직접 처리하며, 앱 내 배송 시스템은 제공하지 않습니다.
