# flower-market 프로젝트 — Claude 가이드

## 필수 읽기
대화 시작 시 반드시 `DESIGN.md`를 읽고 디자인 시스템·컬러·컴포넌트 규칙을 숙지할 것.

## 기술 스택
- Expo + React Native (expo-router)
- Supabase (DB, 인증)
- NativeWind + Gluestack UI (스타일)
- 포트원 (결제)

---

## oh-my-claudecode 명령어 (OMC 설치된 경우)

문제가 생겼거나 기능을 만들 때 아래 명령어를 대화창에 그대로 입력하면 됩니다.

| 상황 | 입력할 말 | 설명 |
|------|----------|------|
| 새 화면/기능 통째로 만들고 싶을 때 | `autopilot` | 기획만 말하면 알아서 전체 구현 |
| 오류가 안 고쳐질 때 | `ralph` | 될 때까지 반복해서 수정 |
| 여러 파일 동시에 수정할 때 | `ultrawork` | 병렬로 빠르게 처리 |
| 버그 원인을 못 찾을 때 | `trace` | 원인 추적 전문 에이전트 |

### 사용 예시
- "autopilot — 꽃 사전 화면 만들어줘"
- "ralph — 로그인 오류 계속 나는데 고쳐줘"
- "trace — 결제 후 주문 목록에 안 뜨는 원인 찾아줘"

---

## 주의사항
- 결제(포트원) / 인증(로그인·로그아웃) 관련 수정은 **반드시 변경 내용 설명 먼저, 승인 후 실행**
- 파일 하나 수정 시 영향받는 다른 파일도 함께 검토 후 안내
